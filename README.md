# ABLY 채용 사전 과제
* 모바일 타겟 가정 너비(최소 320px 최대 768px)로 개발
* A + B + C 모두 구현

## 사용 기술
* Editor: VSCode
* Framework: Vue.js v2.6(Typescript v3.9)
* Library: ElementUI, axios, moment, vue-router, vue-property-decorator
