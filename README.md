# ABLY 채용 사전 과제

## 사용 기술
* Editor: VSCode
* Framework: Vue.js v2.6(Typescript v3.9)
* Library: ElementUI, axios, vue-router, vue-property-decorator

## 구현 범위 (A + B + C)
디자인 가이드는 따로 주어지지 않아, 에이블리 서비스를 참고하여 구현하였습니다.😘

* A. 비밀번호 재설정 페이지
  * 인증 코드 발급 요청 페이지
  * 인증 코드 검증 페이지
  * 비밀번호 변경 페이지

* B. 로그인 페이지

* C. 회원 정보 조회 페이지

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
